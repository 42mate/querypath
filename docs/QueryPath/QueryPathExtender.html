Class QueryPathExtender

Description
Description | Vars (details) | Methods (details)

Implements interfaces:

  • QueryPath

The extension class for Query Path.

This class is used to build extensions to the Query Path library. To create a new custom Query Path extension, create a new class that extends this, add your own methods, and then register it. The extension will then be automatically available.
